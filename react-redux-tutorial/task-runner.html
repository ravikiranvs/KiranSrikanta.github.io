"scripts": { "dev": "webpack-dev-server --content-base src/client/public --inline --hot --port 9000", "lint": "eslint src/client/app/**/*.js", "lint:watch": "esw src/client/app/**/*.js -w" }, ... ``` #####How to run a script command ``` [project]> npm run {script name} ``` ###Running tasks in parallel Install npm-run-all ``` [project]> npm i -D npm-run-all ``` Add script to run tasks in parallel ``` javascript /** [project]\package.json **/ ... "start": "npm-run-all --parallel dev lint:watch", ... ``` This script command will keep hot reloading and linting running. Run the `start` script ``` [project]> npm start ``` <br /> [< Prev](./htrld-eslnt.html) | [Home](./) | [Next >](./routing.html)
