``` Remember to build again to see your changes ``` [project]> node_modules\.bin\webpack ``` ###child components Add a new header component ``` javascript /** [project]\src\client\app\common\components\header.js **/ import React from 'react'; class Header extends React.Component { render () { return ( <div className="header"> {this.props.message} </div> ); } } export default Header; ``` `<App>` to have a child `<Header>` component ``` javascript /** [project]\src\client\app\main.js **/ ... import Header from './common/components/header' ... <div className="app-main"> <Header message={this.props.message} /> </div> ... ``` ###State Let the `<App>` maintain state and pass it as props to `<Header>` ``` javascript /** [project]\src\client\app\main.js **/ import React from 'react'; import {render} from 'react-dom'; import Header from './common/components/header' class App extends React.Component { constructor() { super(); this.state = { message: 'Hello World' } } render() { return ( <div className="app-main"> <Header message={this.state.message} /> <input type="text" value={this.state.message} /> </div> ); } } render( <App></App>, document.getElementById('app') ); ``` Build the app and check. **Notice something wrong!** ###Canging the State Use the textbox's change event to change the `<App>`'s state. ``` javascript /** [project]\src\client\app\main.js **/ ... onChange(event) { this.setState({ message: event.target.value }); } ... ... <input type="text" value={this.state.message} onChange={this.onChange} /> ... ``` Build the app and check.
