What we'll be learning: `react` `redux` `bable` `webpack` `thunk` `eslint` `karma` `mocha` `chai` `sinon` ##What we'll need [<img src="./assets/vscode.png" alt="VS Code" height="200px" width="200px"></img>](https://code.visualstudio.com/Download) [<img src="./assets/nodejs.png" alt="nodejs" height="200px" width="400px"></img>](https://nodejs.org/en/download/) ##How to read the instructions ###Console Commands ``` [project]> foo.exe --bar ``` Here the the command `foo.exe --bar` is executed in the root of your `project` directory. ``` [project]\app> foo.exe --bar ``` Here the the command `foo.exe --bar` is executed in the `app` folder of your `project` directory. ###File Content ``` javascript /** [Project]\src\client\app\main.js **/ function HelloWorld() { const message = "Hello World"; alert(message); } ``` The first comment will have the location of the file.
